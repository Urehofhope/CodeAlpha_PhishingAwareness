# Phishing Awareness Training

A cybersecurity awareness presentation designed to educate 
employees/users on how to identify and avoid phishing attacks.

Created as part of my CodeAlpha Cybersecurity internship.

---

## Topics Covered

- What phishing is and how it works
- Types of phishing attacks
- How to spot a phishing email
- What to do when you receive one
- Real examples and red flags
